# Projeto Instagram Clone — Fase 1

## Objetivo
Construir um Instagram Clone funcional com as telas essenciais, backend em Express + Supabase e deploy em Vercel (frontend) e Railway (backend).

## Entregáveis da Fase 1
- Autenticação de usuários (login/cadastro)
- Feed de posts com curtidas
- Perfil de usuário com grade de posts
- Mensagens diretas (DM)
- Criar post com upload de imagem
- API REST com Supabase e autenticação JWT
- Deploy do frontend e backend

## Organização das subtarefas
1. Frontend
   - Login
   - Cadastro
   - Home (feed)
   - Perfil
   - Mensagens
   - Modal de criação de post
   - Navbar global
2. Backend
   - Configurar Supabase
   - Criar tabelas SQL
   - Implementar rotas de auth
   - Implementar rotas de posts
   - Implementar rotas de usuários
   - Implementar rotas de mensagens
3. Deploy
   - Vercel para frontend
   - Railway para backend

## Prioridade
1. Estruturar backend e conexão com Supabase
2. Autenticação e rotas de usuário
3. Feed e posts
4. Perfil e mensagens
5. UI e navegação
6. Deploy e ajustes finais
